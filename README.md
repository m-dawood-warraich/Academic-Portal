# Academic Portal – Student Management System

A complete desktop application for students, faculty, and admin, built with Python Tkinter and SQLite. Features include course registration, grade viewing, attendance tracking, timetable, live news scraping from UMT, and scholarship alerts from government and international sources.

## Features
- Role-based login (Student, Faculty, Admin)
- Student dashboard: academic summary, courses, grades, attendance, timetable, assignments, support requests, live news, scholarship alerts
- Faculty dashboard: manage courses, mark attendance, enter grades, create assignments
- Admin dashboard: manage students, faculty, courses, view reports
- Live web scraping: UMT news, MOFEPT scholarships, HEC scholarships, TopUniversities scholarships
- Clickable scholarship links open original websites

## Technologies
- Python 3.x
- Tkinter (GUI)
- SQLite (database)
- Requests & BeautifulSoup (web scraping)
- Git & GitHub (version control)
